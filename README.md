# 🗺️ Trippie
**Plan smarter. Travel better.**
 
Trippie je cross-platform aplikácia na spoločné plánovanie výletov. Umožňuje používateľom vytvárať tripy, pridávať aktivity, pozývať priateľov a spravovať celý cestovateľský plán na jednom mieste - bez chaosu v skupinových chatoch.


## ✨ Features
- 🗺️ Vytváranie a správa tripov (PLANNING → ACTIVE → FINISHED)
- 📋 Day-by-day itinerár s aktivitami, časmi a miestami
- 👥 Kolaboratívne plánovanie — pozvi priateľov cez QR kód alebo link
- 👑 Role v tripe — Trip Manager vs. Member
- ✈️ Správa letov a ubytovania (Bookings Hub)
- 🗾 Interaktívna mapa s pinmi aktivít
- 📍 Real-time zdieľanie polohy členov (WebSocket)
- ❤️ Obľúbené miesta
- 🔍 Vyhľadávanie letísk a miest

 
## 🏗️ Tech Stack
 
### Backend
- ASP.NET Core (C#)
- REST API + WebSocket
- Entity Framework Core (Code First)
- PostgreSQL (UUID primárne kľúče)
- JWT autentifikácia
 
### Frontend
- Flutter (cross-platform: iOS & Android)
- Dart
- Google Maps integration
